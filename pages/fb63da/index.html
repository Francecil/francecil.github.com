Windows 下快捷启动应用的小技巧

测试： Win 7

其实 Win 10 就有这个功能

桌面应用太多，找半天？

  1. D 盘下创建 path 目录
  2. 然后将 D:/path 加入 path 环境变量
  3. 把要用的快捷启动拉进该目录，要用的时候就 win+r 运行输入快捷启动名

一般快捷启动名就弄短的好记的就ok了
